# @kubesightapp/animate

This package contains stuff related to creating Lens-applications. 

## Usage

```sh
npm install @kubesightapp/animate
```

```typescript
import { animateFeature } from "@kubesightapp/animate";
import { animateFeature, requestAnimationFrameInjectable } from "@kubesightapp/animate";
import { Animate } from "@kubesightapp/animate";
import type { AnimateName } from "@kubesightapp/animate";
import type { RequestAnimationFrame } from "@kubesightapp/animate";
```
