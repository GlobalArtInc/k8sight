# @kubesightapp/logger

This package contains stuff related to creating Lens-applications. 

## Usage

```sh
npm install @kubesightapp/logger
```

```typescript
import { loggerFeature } from "@kubesightapp/logger";
import type { Logger } from "@kubesightapp/logger";
import { loggerInjectionToken } from "@kubesightapp/logger";
import { winstonLoggerInjectable } from "@kubesightapp/logger";
import { prefixedLoggerInjectable } from "@kubesightapp/logger";
```
